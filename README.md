# NIL — Already Enough

A static site for **NIL**, the fictional wellness brand from the MSc thesis
*Già abbastanza / Already Enough — Self-Improvement Applications and the Manufacture of Deficit*
(Efe Eren Can Bakir, MSc Design, Politecnico di Milano).

NIL is a brand of nothing. It sells no products. The packaging is the critique.

## Pages

- `index.html` — home
- `products.html` — five empty products
- `manifesto.html` — the two-line manifesto
- `about.html` — the unattributed founder

## Type & colour

- Display: EB Garamond · Body: Inter (both via Google Fonts)
- Ground `#FBFAF7` · ink `#1A1A18` · parchment `#F5F0E5` · stone `#CFC4B0` · grey `#6E6E6E`
- No accent colour, on principle.

Plain HTML and CSS. No build step. Served with GitHub Pages.
